# RevMail

RevMail is a modern email platform that reimagines traditional email by introducing features such as editable emails, recall, version history, and real-time synchronization.

Unlike SMTP-based email, RevMail stores and synchronizes messages through a centralized architecture, enabling users to manage sent emails while maintaining complete version history.

## Features

- Authentication
- Compose Email
- Inbox
- Sent Mail
- Drafts
- Edit Sent Emails
- Recall Emails
- Version History
- Attachments
- Read Receipts
- Search
- Real-time Notifications
